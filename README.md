# Trabajo_Practico_Integrador_Programacion_2
Trabajo Integrador de la materia Programación 2, Tecnicatura Universitaria en Programación, Universidad Tecnológica Nacional.

El proyecto desarrollado en Java es un sistema que permite gestionar un catalogo de productos, administracion de usuarios y procesamiento de pedidos detallados, permitiendo crear, editar y eliminar en las diferentes categorias que lo componen.

DESCRIPCION DEL SISTEMA--

Se detalla un sistema de pedidos FOOD STORE , compuesto por un menu que permite administrar categorias, productos, usuarios y pedidos mediante una aplicacion de consola.

El sistema está dividido en módulos basados en Historias de Usuario (HU):

Épica 1: Gestión de Categorías dministración del Catálogo: Creación, edición y listado de categorías para organizar los productos. Validación de Integridad: Control por ID para asegurar que un producto solo pueda asociarse a una categoría existente y activa.

Épica 2: Gestión de Productos Crear, Editar y Listar. Control de nombres, validación de precios y stock.

Épica 3: Gestión de Usuarios Administración de Clientes: Creacion de usario con id unico para asociar a pedidos. Registro de datos con validaciones.

Épica 4: Gestión de Pedidos y Detalles. Creacion de pedidos: Vinculacion de usuario activo y carga a detalles de productos. Calculo del total mediante precio subtotal.

FUNCIONALIDADES-- Cada modulo permite realizar las siguientes operaciones.

1.Listar
2.Crear
3.Editar
4.Eliminar
5.Volver al menu principal

EJECUCION--

1.Abir el proyecto NetBeans.
2.Ejecutar la clase 'Main'.
3.Utilizar el menu en consola para gestionas las opciones.
CONCEPTOS APLICADOS-- -Encapsulamiento -Herencia -Polimorfismo -Manejo de excepciones -Validacion de datos -Colecciones
